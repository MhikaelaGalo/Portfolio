## Projects

### 1\. Aegis: Security Verification Platform

  * **Description:** An award-winning identity protection platform (2nd Place, Hackathon Jam 2026) using Zero-Knowledge (ZK) proof concepts to verify users without exposing sensitive data.
  * **Tech Stack:** Next.js, React, JavaScript, CSS-in-JS.


### 2\. Pawlax: Pet Services Management

  * **Description:** A full-stack booking system for pet owners and administrators, featuring real-time scheduling, account management, and a custom-built admin dashboard.
  * **Tech Stack:** PHP, MySQL, HTML, CSS, JavaScript.


### 3\. Plug and Play: E-commerce Backend

  * **Description:** A robust RESTful API for modern e-commerce, handling role-based authentication (RBAC), secure product uploads, and dynamic inventory tracking.
  * **Tech Stack:** Node.js, Express.js, MongoDB, JWT, Bcrypt, Multer.


### 4\. Heart Attack Risk Prediction

  * **Description:** A predictive analytics model using binomial logistic regression to identify high-risk heart attack outcomes with 88% accuracy.
  * **Tech Stack:** R, Logistic Regression, Data Cleaning.
 

### 5\. Choice Story: Narrative Engine

  * **Tech Stack:** Socket.io, Node.js, JavaScript (ES6+).
